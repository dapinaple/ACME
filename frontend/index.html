ACME Clipper

Clip coupons for your grocery list

Works with your normal ACME login

ACME doesn't allow third-party apps to sign in with phone verification. Instead, this tool runs inside Safari on acmemarkets.com while you're already logged in — the same way the website and app work.

Best for iPhone users (including app-only accounts)

No password needed for our app. Sign in on ACME's website with your phone number like you normally do, then use the clipper panel there.

  1. Install the free Userscripts app from the App Store.
  2. In Userscripts, tap + → From URL and paste:
    Loading script URL...
  3. Enable the script, then open ACME Coupons & Deals in Safari.
  4. Sign in with your phone number on ACME's site (text verification works here).
  5. Tap the red A button → enter your grocery list → Clip Matching Coupons.

    Why doesn't sign-in work in this app?

    ACME uses private login systems for their app and website. Third-party tools can't use phone-text login, and ACME has disabled the old password API this project originally used. Running the clipper directly on acmemarkets.com avoids that problem entirely.
